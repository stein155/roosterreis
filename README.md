# roosterreis
Een schoolopdracht om kennis te maken met een functionele programmeertaal, waarbij zelf een 'challenge' verzonnen mocht worden.
Als challenge heb ik gekozen om automatisering toe te passen omtrent mijn lesrooster en de 9292 API. Deze tool bekijkt, op basis van de eerste lestijd, voor de volgende dag welke trein(en) ik moet nemen om op de HAN (Arnhem) te komen.

Binnen deze opdracht heb ik onder andere kennisgemaakt met de parsing mogelijkheden die Haskell biedt, onder andere om JSON-data uit een API om te zetten tot leesbare gegevens. Daarnaast ook het uitvoeren van HTTP-aanvragen en het converteren van datums.
