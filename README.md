# roosterreis
Een schoolopdracht om kennis te maken met een functionele programmeertaal, waarbij zelf een 'challenge' verzonnen mocht worden.
Als challenge heb ik gekozen om automatisering toe te passen omtrent mijn lesrooster en de 9292 API. Deze tool bekijkt voor de volgende dag welke trein(en) ik moet nemen om op de HAN (Arnhem) te komen.
